# School Management Schedule Frontend

Frontend-модуль расписания учебных занятий для LMS-платформы.

Модуль предназначен для отображения учебных событий в календарном формате. Он помогает пользователям быстро понимать, какие занятия запланированы, когда они проходят, к какому направлению относятся и кто является преподавателем.

## Назначение проекта

Расписание является одной из основных точек входа в учебный процесс. Вместо просмотра занятий через внешние таблицы или отдельные списки пользователь получает календарный интерфейс, в котором учебные события отображаются по датам.

Модуль решает две основные задачи:

- для студентов и преподавателей — просмотр учебных занятий в удобном календарном формате;
- для администратора — базовое управление расписанием: создание, редактирование, удаление и перенос занятий.

## Основные возможности

В рамках модуля предполагаются следующие сценарии:

- просмотр расписания в календарном виде;
- переключение периода расписания;
- отображение карточек занятий внутри дат;
- фильтрация занятий по направлениям;
- просмотр основной информации о занятии;
- создание занятия администратором;
- редактирование занятия;
- удаление занятия;
- перенос занятия на другую дату.

## Структура проекта

Основные части проекта:

```text
src/
  app/        # инициализация приложения, роутинг, провайдеры
  pages/      # страницы приложения
  modules/    # функциональные модули
  entities/   # бизнес-сущности
  shared/     # общая инфраструктура, компоненты, API-клиент
```

Основной модуль проекта:

```text
src/modules/schedule
```

Он содержит код, связанный с отображением расписания, календарной сеткой, карточками событий, фильтрами, выбором периода и вспомогательной логикой для работы с датами.

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

Проверить типы:

```bash
npm run typecheck
```

Запустить линтер:

```bash
npm run lint
```

Проверить форматирование:

```bash
npm run format:check
```

Собрать проект:

```bash
npm run build
```
